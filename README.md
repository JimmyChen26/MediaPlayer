# 多媒體播放器

本專案是一個使用 **C# Windows Forms** 製作的簡易多媒體播放器，透過 **Windows Media Player 控制項** 播放影片，並提供基本的瀏覽、播放、暫停與停止功能。

## 專案功能

本程式具備以下功能：

1. 瀏覽本機影片檔案
2. 播放選取的影片
3. 暫停正在播放的影片
4. 停止影片播放
5. 顯示目前選取的檔案名稱
6. 顯示操作說明
7. 自動調整播放器與按鈕位置
8. 具有簡潔美觀的版面配置

## 使用技術

- C#
- Windows Forms
- .NET Framework
- Windows Media Player ActiveX 控制項

## 開發環境

建議使用以下環境開發與執行：

- Visual Studio 2022
- Windows 作業系統
- Windows Forms App (.NET Framework)
- Windows Media Player 元件
